# First Exam Project

##  Опис проєкту

Цей проєкт — фінальна робота, створена на основі макету. Мета — побудувати адаптивний сайт з чистою структурою HTML, організованими стилями SCSS та сучасним підходом до верстки.
---

## Підготовка

### 1. Створення репозиторію

- Створила GitHub репозиторій для зберігання проєкту.

### 2. Аналіз макету

- Проаналізувала макет та зробила нотатки на папері.
- Обрала підхід **desktop-first**.
- Визначила основні секції, заголовки та картинки.
- Продумала використання респонсивних зображень.
- Визначила головні змінні для стилів.

### 3. Підготовка шрифтів

- Знайшла потрібні шрифти в інтернеті та завантажила їх.
- Конвертувала **.ttf** у **.woff** та **.woff2**.
- Додала шрифти до проєкту.
- Підключила їх у файлах `fonts.scss`, `vars.scss`, `mixins.scss` та `main.scss`.

### 4. Підготовка медіа

- Оптимізувала та завантажила зображення у проєкт.
- Створила **sprite.svg** для іконок.
- Підключила **favicon**.
- Додала відео у папку проєкту.

---

## Старт розробки

### 1. Створення структури проєкту

- Організувала папки та файли для зручної роботи.

### 2. HTML структура

- Створила секції сторінки та додала відео.
- Написала розмітку для всіх елементів.
- Заміняла `<picture>` на `<img>`, де це було доцільно.
- Видалила зайві контейнери.

### 3. Стилізація та адаптивність

#### Загальні стилі

- Додала контейнер та базові стилі.
- Оформила хедер, меню та бургер-меню.
- Задала стилі для заголовків, параграфів та відео.
- Додала падінги до хедера.

#### Кнопки

- Створила стилі для кнопок.
- Додала псевдоелементи та ефекти при **hover**.

#### Секції

**Hero**

- Налаштувала бекграунд, щоб зображення не обрізалось.
- Додала медіа-правила для адаптивності.

**Quote**

- Стилізувала бекграунд та текст.

**Info**

- Замінила `grid` на позиціонування елементів.
- Виправила відступи та додала псевдоелементи.
- Створила адаптивне меню:

  - Зменшила шрифти.
  - Додала псевдоелемент для третього елемента.
  - Налаштувала **row** для мобільної версії.
  - Додала анімації для секції та курсивного тексту.

**Benefits**

- Замінила `570px` на `1fr` для гнучкості верстки.
- Додала адаптивні стилі.

**Reviews**

- Додала картки з відгуками та бекграунд.
- Для адаптиву створила додаткову картку.

**Promises**

- Вирівняла кнопку на мобільній версії.
- Змінила картинку на бекграунд.
- Задала білий текст.

**Contact**

- Стилізувала контейнер, форму та інпути.
- Налаштувала стилі для плейсхолдерів.
- Додала `wrap` та `gap` для адаптивності.

**Footer**

- Додала класи та стилі.
- Створила стилі для посилань та ефекти **hover**.

## Виправлення та оптимізація

- Виправила баги у кнопках та формі.
- Налаштувала відео-контейнер та додала керуючі кнопки.
- Вирівняла відображення цін у доларах.
- Виправила **hover** для псевдоелементів кнопок.
- Виправила баг у секції **Info** для коректного відображення разом із **Hero**.
- Прибрала `position: fixed` у хедері для мобільних пристроїв.
- Додала анімацію відкриття мобільного меню.

## Підсумок

У проєкті реалізовано:

- Чиста, організована HTML-структура.
- SCSS зі змінними, міксинами та структурованими файлами.
- Адаптивна верстка для різних пристроїв.
- Оптимізовані зображення та шрифти.
- Анімації та інтерактивні елементи.
